Download section of the International Chicory Database

All available passport data of accessions included in the International Chicory Database are downloadable as zipped Excel file. This zip file also contains the decoding tables, except for the FAO institutes table that is available on the Wiews website (PGR > Download).
Characterization and evaluation data generated within the framework of the EU GENRES project entitled "Leafy vegetables germplasm, stimulating use" (AGRI-2006-0262) are also downloadable as zipped Excel file.
